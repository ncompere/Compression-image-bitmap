/!\ Ne pas renommer Main.jar et le laisser dans le même dossier que le dossier pngs /!\

Contenu du Dossier :
- Main.jar : Le programme compilé et exécutable
- Code Source : Dossier contenant le code source
- projet.pdf : sujet
- pngs : Dossier contenant les images de tests fournies avec le sujet

Instructions :
- (Facultatif) : Ajouter des images dans le dossier pngs
- Commande pour démarrer en mode interactif : "java -jar Main.jar"
- Commande en mode non-interactif : "java -jar Main.jar nomFichier delta Phi"
			Exemple : java -jar Main.jar 512-books.png 50 30000
			nomFichier doit être dans le dossier pngs et doit être différent de "temp.png" (nom reservé)
